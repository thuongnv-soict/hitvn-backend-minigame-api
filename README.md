# MiniGame - API

API for mini game of Hitvn

## Getting Started


### Installing

Running api:

```bash
$ go run main.go
```

Running consumer lottery result daily:

```bash
$ go run lottery.go
```

## Built With
* [Golang](https://golang.org/) - The programming language used
* [Go Echo](https://echo.labstack.com/) - The Go web framework used
* [Redis](https://redis.io/) - Cache
